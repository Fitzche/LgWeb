CharactUHC — Rôles JSON
Rôles personnalisables · Jauges · Conditions · Actions imbriquées
CharactUHC — rôles définis en fichiers .json. Chaque rôle a ses stats, pouvoirs, jauges et commandes. Plusieurs listes se cumulent (un joueur peut être "Mage Elfe").
📁 Fichiers
Gamemodes/monJeu.json
roles/guerrier.json
Lancer : /lg startCharact monJeu.json
✅ Démarrage
1. Créez votre gamemode
2. Créez vos rôles
3. /lg startCharact [nom.json]
4. Test : /lga jsonTest roles/mage.json

Items Spéciaux
Tous les items gérés par SpecialItemHolder (hors items "Inf" de test) · Cliquez sur un item pour le détail
Ces items constituent l'inventaire spécial des joueurs (specialItemsOwned dans PlayerData). Pour ajouter un nouvel item : ajoutez une entrée dans les deux tableaux SPECIAL_ITEMS_DEF et ITEMS_CATALOG dans ce fichier HTML (ils sont commentés et faciles à trouver), puis créez l'item dans SpecialItemHolder.java.
➕ Ajouter un nouvel item
1. Dans SPECIAL_ITEMS_DEF (pour l'affichage en chip dans le profil) :
'MonNouvelItem': { icon:'⚗️', label:'Mon Item', color:'#34d399', bg:'rgba(52,211,153,.12)', border:'rgba(52,211,153,.35)' },
2. Dans ITEMS_CATALOG (pour la fiche dans cette page) :
{ key:'MonNouvelItem', icon:'⚗️', name:'Mon Item', color:'#34d399', border:'rgba(52,211,153,.35)', mode:'Mode de jeu', tags:['Tag1','Tag2'], desc:'Description de l\'effet de l\'item.', howToGet:'Conditions d\'obtention.' },
3. Dans SpecialItemHolder.java, ajoutez la gestion de l'item dans onPlayerInteract().
